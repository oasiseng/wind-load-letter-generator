# Contributing

This project is published as a public reference implementation under the MIT license. It is **not** an actively maintained open-source project, and the contribution policy reflects that.

## What is accepted

- **Forks.** You are free to fork this repo and modify it for your own purposes under the terms of the [MIT License](LICENSE). No notification or permission needed.
- **Issue reports for documented bugs**, where "bug" means a clear deviation from documented behavior with a reproducible example. These will be read but not necessarily acted on. No timeline is offered.

## What is not accepted

- **Pull requests.** PRs will be closed without review. The codebase is small enough that it's faster to discuss the change in an issue than to review a PR.
- **Feature requests.** The scope of v1.0 is intentionally narrow (single-family residential, single-story, simple geometry). Requests to extend the scope are out of scope until v2.0. There is no public roadmap or release schedule.
- **Support requests.** This is not a supported product. The README, code comments, and inline calc walkthrough are the documentation. There is no Discord, no Slack, no mailing list.
- **Questions about engineering judgment, code interpretation, or specific projects.** The tool produces a template; an engineer's job is to verify the output and apply judgment to the specific project. We do not provide that service through GitHub.
